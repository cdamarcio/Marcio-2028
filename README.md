# 🏛️ Márcio: O Fiscal do Povo - 2028

Landing page oficial da iniciativa **"Fiscal do Povo"**, um projeto focado em transparência, fiscalização de recursos públicos e combate à corrupção no município de **Conceição do Araguaia (CDA)**.

---

## 🚀 O Projeto
Este site foi desenvolvido para servir como canal direto entre o cidadão e a fiscalização municipal. Aqui, a população pode acompanhar denúncias reais, verificar o status de resoluções de problemas urbanos e conhecer propostas de melhoria para a cidade em 2028.

### ✅ Funcionalidades
* **Painel de Denúncias:** Visualização de irregularidades em licitações, infraestrutura e saúde.
* **Status em Tempo Real:** Acompanhamento se uma denúncia está "Pendente", "Em Investigação" ou "Resolvido".
* **Propostas Estratégicas:** Plano de ação para segurança, educação, saúde e geração de empregos.
* **Canal Direto:** Integração com WhatsApp para recebimento de novas denúncias.

## 🛠️ Tecnologias Utilizadas
O projeto foi construído com foco em performance e responsividade (Mobile First):
* **HTML5:** Estrutura semântica.
* **CSS3:** Estilização moderna com variáveis.
* **JavaScript:** Animações de entrada e interatividade.

## 📂 Estrutura do Repositório
* `index.html`: Estrutura principal do site.
* `style.css`: Estilização e design responsivo.
* `script.js`: Lógica de animações.
